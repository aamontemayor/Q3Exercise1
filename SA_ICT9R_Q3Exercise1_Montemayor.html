HTML

HyperText Markup Language

HTML is a markup language or a system used to create web pages in the internet. Designed by the scientist Tim Berners-Lee during the 1980s (Hemmendinger, 2023). The web pages designed contain links that can retrieve other related web pages. HTML has tags and elements used to create the page.

Hemmendinger, D. (2023). HTML | Definition & Facts | Britannica. In Encyclopædia Britannica. https://www.britannica.com/technology/HTML

CSS

Cascading Style Sheets

CSS is a stylesheet language used to decorate or to beautify your web page. There are three types of CSS, inline, internal, and external CSS. CSS can be used to change the color, size, space, font, and any other style for your webpage

Bootstrap

Bootstrap Version 4

According to W3Schools, Bootstrap is a popular CSS framework designed to easily create responsive web pages. The creation of websites and web apps mostly depend on Bootstrap. The responsive web page is designed to automatically adjust the page and make it look good.
